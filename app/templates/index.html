project introduction

This assignment was completed by Yuqiang Hu and Zhuang Huang together. Since Huang Zhuang's computer broke down, all the codes were uploaded to GitHub by Hu Yuqiang. We worked together happily. This project is to do a small sudoku game, we made a 9x9 grid sudoku game mainly use front-end JavaScript to make this shudu game. And then use Ajax in index.js to transfer user records to the back end, and stored in the database. Then we implemented a ranking feature where we pulled out the records the user had stored in the database. We also rank users according to how much time they spend, so as to motivate them to play our games more and compete for the TOP1. In order to make the game more interesting and difficult at the same time, we set up three kinds of difficulty (easy, middle, hard), new users can choose the Easy mode to experience our game faster, old users can select the mode of more difficult to increase the challenge the difficulty, continue to attract more users to play our game. We do this by setting math.random () > chance in the index.js file; Chance adjusts the difficulty of the game by setting different thresholds to show different numbers of squares, changing the number of squares the game needs to fill. And to keep this game always new for our users, our Sudoku games are randomly generated every time, and every time is different.
